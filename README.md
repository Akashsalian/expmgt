# 💰 Expense Tracker Web App

## 📌 Overview
FinTrack Pro is a full-stack web application designed to help users efficiently track and manage their income and expenses. Built using Flask and MySQL, the application provides real-time financial insights with a clean and modern user interface.

---

## ✨ Features
- ➕ Add income and expense transactions  
- 📋 View all transactions in a structured table  
- ✏️ Edit and update existing records  
- 🗑️ Delete transactions  
- 💰 Real-time balance calculation  
- 📊 Dashboard with income, expense, and balance overview  
- 🎨 Modern UI with animations and gradient effects  

---

## 🛠️ Tech Stack
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS  
- **Database:** MySQL  
- **Tools:** VS Code, Git, GitHub  

---

## ⚙️ How It Works
1. User adds income or expense through the form  
2. Data is stored in MySQL database  
3. Application fetches and displays transactions  
4. Balance is calculated dynamically  
5. Users can edit or delete records anytime
